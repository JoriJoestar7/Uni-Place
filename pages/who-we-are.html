Proyecto universitario con propósito

Quiénes somos

Somos un equipo de estudiantes de Negocios Digitales que desarrolla UniPlace como una solución tecnológica para una necesidad real de la comunidad universitaria: encontrar servicios confiables y dar visibilidad a emprendimientos locales.

Nuestra visión

Queremos que UniPlace funcione como una herramienta útil, clara y cercana para estudiantes, emprendedores y administradores.

01

Diseño con sentido

Creamos una interfaz minimalista, oscura y ordenada para que la experiencia sea cómoda, moderna y sin ruido visual.

02

Tecnología aplicada

Combinamos frontend, backend, base de datos e inteligencia artificial para resolver una necesidad concreta.

03

Impacto local

Buscamos fortalecer la conexión entre la comunidad universitaria y los emprendimientos cercanos.

Lo que construimos

UniPlace pasó de ser una idea de recomendación a convertirse en una web app con roles, base de datos, aprobación administrativa y chat conectado a conocimiento propio.

Para estudiantes

Un asistente que permite preguntar por servicios cercanos, comida, papelería, transporte o negocios útiles, reduciendo el tiempo de búsqueda y entregando opciones más relevantes.

Para emprendedores

Un espacio donde pueden registrar su negocio, detallar productos, precios, horarios y datos clave para que el sistema pueda recomendarlos cuando realmente encajan con la consulta del usuario.

Buenas prácticas

El proyecto considera seguridad, trazabilidad y organización de datos como parte del desarrollo.

Control de acceso

Registro, login y roles para separar estudiantes, emprendedores y administradores.

Integridad de datos

Información estructurada en MySQL para evitar duplicidad y mantener registros ordenados.

Trazabilidad

Estados de aprobación, razones de rechazo y registros de recomendación para seguimiento del sistema.

Hacia dónde vamos

UniPlace está preparado para crecer con nuevas funciones sin romper su estructura principal.

01

Edición de emprendimientos

Permitir que los negocios actualicen datos, menús, precios y promociones.

02

Reportes estadísticos

Mostrar consultas frecuentes, negocios recomendados, usuarios activos y tendencias.

03

Bitácoras y auditoría

Registrar acciones importantes para control, evidencia y mantenimiento del sistema.
