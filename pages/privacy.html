Protección de datos

Privacidad

Esta política explica cómo UniPlace maneja la información dentro de esta versión inicial. Actualmente, la aplicación funciona como una maqueta web con almacenamiento local en el navegador.

1. Información que puede almacenarse

En esta versión, UniPlace puede guardar datos como nombre de usuario, correo ingresado, conversaciones simuladas y preferencias básicas dentro del navegador mediante localStorage. Esta información se guarda en el dispositivo del usuario.

2. Información sensible

No se recomienda ingresar contraseñas reales, números de identificación, datos bancarios, información médica, documentos privados, información institucional confidencial o cualquier dato que pueda considerarse sensible.

3. Conversaciones

Las conversaciones guardadas sirven para simular una experiencia tipo chatbot. En una versión futura con integración real de IA, será necesario explicar con claridad si los mensajes se envían a un servidor, API externa o proveedor de inteligencia artificial.

4. Cookies

Actualmente UniPlace no utiliza cookies. En el futuro, si se agregan cookies para analítica, sesiones, personalización o funcionamiento técnico, se deberá añadir una política de cookies y un aviso claro para el usuario.

5. Control del usuario

El usuario puede borrar sus conversaciones desde el dashboard usando la opción de limpiar datos. También puede borrar manualmente los datos del navegador desde la configuración de su dispositivo.

Nota para desarrollo futuro: cuando UniPlace conecte una API real, base de datos, autenticación o cookies, esta política deberá actualizarse antes de publicarse oficialmente.
