Créditos del proyecto

UniPlace fue desarrollado como una propuesta de sistema de recomendación inteligente para la comunidad universitaria.

Área técnica

Desarrollo web con HTML, CSS, JavaScript, Node.js, Express, MySQL, autenticación con JWT y base de conocimiento para recomendaciones.

Área estratégica

Enfoque en conexión universitaria, visibilidad para emprendimientos locales, organización de información y uso responsable de inteligencia artificial.
