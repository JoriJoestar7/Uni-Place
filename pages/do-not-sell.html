Control de información personal

Como manejamos tu información personal

Esta sección informa al usuario sobre el uso, venta o intercambio de información personal dentro de UniPlace. En esta versión inicial, UniPlace no vende ni comparte información personal con terceros.

1. Estado actual de UniPlace

Actualmente, UniPlace funciona como una aplicación web en desarrollo. Los datos principales, como conversaciones simuladas o información de acceso de prueba, se almacenan localmente en el navegador mediante localStorage.

2. Venta de información personal

UniPlace no vende información personal del usuario. No existe, en esta versión, intercambio comercial de datos personales con anunciantes, brokers de datos, plataformas publicitarias o terceros.

3. Compartir información personal

UniPlace no comparte información personal con terceros en esta etapa del proyecto. Si en el futuro se integra una API externa, una base de datos, servicios de analítica o proveedores de IA, esta página deberá actualizarse para explicar claramente qué datos se envían, con qué finalidad y bajo qué condiciones.

4. Solicitud de exclusión

Si UniPlace llegara a manejar datos reales en el futuro, el usuario debería poder solicitar que su información personal no sea vendida ni compartida. Esta página queda preparada como base para ese mecanismo de control.

5. Futuras actualizaciones

Cuando se agreguen funciones como cuentas reales, cookies, analítica, anuncios, integraciones con IA o almacenamiento en servidor, será necesario actualizar esta política y agregar controles claros de privacidad.

Por ahora, esta sección funciona como una declaración preventiva y transparente. No se agregan cookies todavía. Esa política puede incorporarse más adelante cuando la app realmente las utilice.
