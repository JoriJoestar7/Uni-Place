Cómo funciona

UniPlace organiza el flujo completo de uso: acceso de usuarios, registro de emprendimientos, revisión administrativa, base de conocimiento y recomendaciones inteligentes dentro del chat.

Flujo principal

El sistema conecta la información del emprendimiento con la consulta del estudiante de forma controlada y segura.

01

El usuario accede a UniPlace

La plataforma cuenta con registro e inicio de sesión para estudiantes, emprendedores y administradores. Cada rol tiene permisos diferentes dentro del sistema.

02

El emprendimiento registra su información

El emprendedor completa una ficha con datos del negocio, menú, precios, horarios, ubicación, métodos de pago, preguntas frecuentes y palabras clave útiles para la IA.

03

El admin revisa y aprueba

Antes de aparecer en las recomendaciones, el negocio pasa por una revisión administrativa. Si es rechazado, el emprendedor puede ver la razón, corregir la información y volver a intentarlo.

04

La base de conocimiento se activa

Cuando el negocio es aprobado, su información pasa a una base de conocimiento interna que el chat puede consultar. Esto permite priorizar emprendimientos registrados y aprobados dentro de UniPlace.

05

La IA recomienda con contexto

Cuando un estudiante pregunta por comida, papelería, transporte u otro servicio, el sistema busca negocios relevantes según categoría, presupuesto, ubicación, horarios y palabras clave.

Módulos del sistema

UniPlace está organizado en módulos funcionales que permiten mantener el proyecto ordenado y escalable.

Página principal

Presenta el proyecto, su enfoque, acceso al registro y páginas informativas.

Autenticación

Gestiona registro, inicio de sesión, roles y protección de rutas mediante token.

Chat inteligente

Procesa preguntas naturales y recomienda negocios según los datos aprobados.

Registro de negocios

Permite ingresar información completa para alimentar la base de conocimiento.

Panel admin

Sirve para aprobar, rechazar, ocultar o revisar emprendimientos registrados.

Base de datos

Almacena usuarios, negocios, horarios, menú, FAQs, conocimiento IA y registros de recomendación.

La recomendación inteligente

El chat no recomienda al azar: cruza la pregunta del usuario con datos estructurados de negocios aprobados.

A

Ubicación y proximidad

Prioriza opciones cercanas a la zona universitaria, campus o punto de referencia indicado.

B

Horario y disponibilidad

Considera días y horas de atención para sugerir negocios útiles en el momento de la consulta.

C

Presupuesto y categoría

Filtra por precios, productos, servicios, palabras clave y necesidades concretas del estudiante.
